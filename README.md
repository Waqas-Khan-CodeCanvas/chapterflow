
# 📚 Book Heaven

A clean, responsive, and production-ready online bookstore built using the MERN stack. Users can browse books, manage favorites and cart items, place orders, and track their purchases, while administrators can manage books and orders through a dedicated dashboard.

## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/Waqas-Khan-CodeCanvas/Book-Heaven-fullstack.git
```

### Install Dependencies

#### Frontend

```bash
cd frontend
npm install
npm run dev
```

#### Backend

```bash
cd backend
npm install
npm start
```

## 🌟 Why Book Heaven?

Book Heaven is a modern MERN stack bookstore project designed to demonstrate full-stack development concepts, including authentication, CRUD operations, state management, MongoDB relationships, and admin functionality. It provides a clean architecture and scalable code structure suitable for learning and portfolio projects.
